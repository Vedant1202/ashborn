# Brand assets

Ashborn's visual identity — a single-tone shadow-flame helm, used as the mark.

| File                                              | What it is                                           |
| ------------------------------------------------- | ---------------------------------------------------- |
| `images/icon.png`                                 | source emblem (transparent) the SVGs are traced from |
| `ashborn.svg`                                     | the mark in `#26103E` — for light backgrounds        |
| `ashborn-inverse.svg`                             | the mark in `#7C5CC4` — for dark backgrounds         |
| `favicon.svg`, `favicon-32.png`, `favicon-16.png` | square favicon                                       |
| `ashborn-hero.svg` / `.png`                       | README hero banner (dark)                            |
| `ashborn-social.svg` / `.png`                     | 1280×640 social / link-preview card                  |

**Palette:** emblem `#26103E`, on-dark `#7C5CC4`, night `#0B0A14`, wordmark `#E8DEF6`.
**Wordmark:** `ashborn` — lowercase, monospace, letter-spaced.
**Tagline:** See what your AI agents access, call, and send.

The PNGs are regenerated from the source with `rsvg-convert`; the SVGs are traced
from `images/icon.png`.

To set the link preview shown when the repo is shared: **Settings → General →
Social preview → upload `ashborn-social.png`**.
