# @ashborn-sec/core

The core contract for [Ashborn](https://github.com/Vedant1202/ashborn): the tool-call event model, findings and risk annotations, the per-session provenance ledger, the tool-definition-drift detector, and the untrusted-data-egress risk annotation.

This is the library layer. Most users want the CLI, [`@ashborn-sec/cli`](https://www.npmjs.com/package/@ashborn-sec/cli), or the benchmark, [`@ashborn-sec/bench`](https://www.npmjs.com/package/@ashborn-sec/bench).

Full docs, results, and methodology: **https://github.com/Vedant1202/ashborn**
